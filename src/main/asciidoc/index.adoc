== 설명
이름(sillog title)을 기준으로 한 조회 구현.

특정 sillog title(행사타이틀) 을 가진 행사들을 모두 조회하는 기능.

같은 행사타이틀을 가진 모든시퀀스 행사들을 리스트 형태로 제공.
